Products — Operations

Internal Dashboards

Build internal tools, operational command centers, and custom views on top of your Ventryx data — without building and hosting a separate admin panel from scratch.

What you can build

  • Operations dashboards showing live workflow health and queue status
  • Customer-facing status pages fed by real platform metrics
  • Finance views showing usage, billing trends, and projected costs
  • Engineering oncall boards with error rates, alerts, and recent incidents
  • Custom approval queues for non-technical reviewers

Dashboard components

ComponentDescription
Metric tilesSingle-value KPIs with trend indicators (up/down vs. prior period)
Time series chartLine or bar charts of any numeric metric over time
Table viewPaginated, searchable lists of resources (workflows, events, tasks)
Status indicatorGreen / yellow / red health indicator with threshold config
Activity feedReal-time stream of recent events or audit entries
Approval queueList of pending approvals with one-click approve/reject

Data sources

Dashboard components can pull from:

  • Ventryx Analytics API (request metrics, workflow stats, event throughput)
  • Your own API endpoints, fetched via configured HTTP data sources
  • Static data or manually entered values for context panels

Sharing and access

Dashboards can be shared with your full organization, restricted to specific roles, or published as read-only external links for stakeholders who don't have a Ventryx login.
